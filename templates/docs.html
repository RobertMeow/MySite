API Documentation

Welcome to my little API documentation. Below you will find a list of available API methods and examples of their use.


Endpoint

Our API endpoint is located at https://api.hella.team. At the moment, the most up-to-date version of the API==2


Authentication

For authorization, you must specify access_token as a parameter, you can get it here. If you do not specify access_token, then the limits will be very strict.


API Methods

  • GetStickers

    Get user stickers. Here's how to use it:

              import requests

url = "https://api.hella.team/method/getStickers"
params = {
    "v": "2",
    "user_id": 1
}

resp = requests.get(url, params=params)
if resp.status_code != 200:
    raise Exception("Error: {}".format(resp.status_code))
print(resp.json())

  • GetSticker

    Get information about the sticker pack, required parameter product_id

              import requests

url = "https://api.hella.team/method/getSticker"
params = {
    "v": "2",
    "sticker_id": 1,
    "product_id": 1
}

resp = requests.get(url, params=params)
if resp.status_code != 200:
    raise Exception("Error: {}".format(resp.status_code))
print(resp.json())
